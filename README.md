# NowShip Live

NowShip Live is a custom CRM and business dashboard for NowShip.

## First version includes

- Dashboard
- Company list
- Add company form
- Supabase database connection

## Setup

1. Copy `.env.local.example` to `.env.local`
2. Add your Supabase project URL and anon key
3. Run `npm install`
4. Run `npm run dev`
